# 🌿 Greenden - Tailwind CSS Website

## 📌 Overview
**Greenden** is a responsive multi-page website built using **HTML** and **Tailwind CSS**.  
It features a clean and modern UI design, focusing on Tailwind’s utility-first approach to create a flexible and attractive layout.

This project helps beginners understand real-world website structure and responsive design techniques.

---

## 🚀 Website Pages
The website contains **3 main pages**:

- **Home Page**  
  Welcomes users with a clean landing section and introduction.

- **Products Page**  
  Displays plant-related products using a responsive grid layout.

- **Contact Page**  
  Includes a structured contact form section for user interaction.

---

## 📱 Responsive Design
- Fully responsive layout for **mobile, tablet, and desktop** screens.
- Built using Tailwind responsive breakpoints:
  - `sm:`
  - `md:`
  - `lg:`

Each section maintains consistent spacing, alignment, and typography for a professional UI.
